# Tagger

## **Why?**  
An easy and intuitive way to tag AWS Resources  

Currently supports:  
* Lambda  
* DynamoDb  
* S3  
* EC2  

## **Requirements**
* AWS CLI installation [(Link)](https://docs.aws.amazon.com/cli/latest/userguide/installing.html)
